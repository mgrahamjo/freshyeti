Intro

Breadbox is an MVC framework for Node.jsÂ®. It is currently in development and should not be used without expectation of breaking changes.

Breadbox strives to provide a broad feature set in one small, simple package. Out of the box, it includes:

  • A template engine
  • A file-based data store
  • Error interceptors to relieve error handling annoyances
  • Security features like automatic HTML escaping and CSRF protection
  • Authentication and session mangement
  • A promise object implementation
  • Support for POST data parsing including file uploads

Some examples in this document use ECMAScript 6, and currently require use of a transpiler such as Babel. While Breadbox uses ES6 internally, there is no requirement that you use it in your app.
